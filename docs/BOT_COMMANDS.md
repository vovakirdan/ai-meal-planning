# Bot Commands

Этот документ фиксирует реальные slash-команды бота и их назначение.
Его можно использовать как внутренний справочник и как основу для описания в BotFather.

## Public Commands

Ниже только те команды, которые уже существуют и доступны пользователю.

### `/start`

Запускает стартовую настройку профиля.

Что делает:

- создает или продолжает onboarding;
- собирает базовые настройки семьи;
- собирает участников семьи;
- настраивает напоминания;
- в конце предлагает перейти к плану недели.

Когда использовать:

- при первом запуске;
- если настройка профиля еще не завершена;
- если нужно заново пройти стартовый сценарий.

Примечание:

- если onboarding уже завершен, бот не запускает его повторно, а предлагает перейти к `/plan`.

### `/plan`

Запускает составление плана на неделю.

Что делает:

- предлагает диапазон дат;
- уточняет шаблон приемов пищи на неделю;
- спрашивает про уклон недели и дополнительные пожелания;
- предлагает учитывать запасы;
- создает черновик недели;
- генерирует меню.

Когда использовать:

- чтобы составить новый план;
- чтобы пересобрать неделю с нуля.

Примечание:

- если уже есть черновик, бот предлагает оставить его или удалить и начать заново.

### `/week`

Открывает текущую неделю.

Что делает:

- показывает текущий активный план;
- если есть черновик, открывает его;
- если черновика нет, открывает последнюю подтвержденную неделю;
- позволяет провалиться в день, прием пищи и карточку блюда;
- позволяет редактировать блюда и подтверждать неделю.

Когда использовать:

- чтобы быстро вернуться к плану;
- чтобы проверить или отредактировать блюда;
- чтобы подтвердить черновик.

Примечание:

- подтвержденная неделя тоже остается редактируемой.

### `/cancel`

Останавливает текущий сценарий с вводом текста.

Что делает:

- очищает активный FSM flow;
- завершает текущий шаг, если пользователь передумал.

Когда использовать:

- если пользователь хочет выйти из текущего сценария;
- если нужно начать шаг заново;
- если бот ждет текстовый ответ, а пользователь хочет отменить действие.

Примечание:

- это техническая команда отмены, а не отдельный продуктовый сценарий.

### `/review`

Запускает ручной сбор обратной связи по блюдам.

Что делает:

- выбор дня из последней подтвержденной недели;
- проход по схеме `блюдо -> участник семьи`;
- оценка блюда;
- optional comment;
- сохранение feedback для следующих генераций.

Когда использовать:

- в конце дня или позже, когда хочется оценить блюда;
- чтобы накопить память по вкусам участников семьи;
- чтобы следующие планы лучше учитывали реальные реакции.

## BotFather Draft

Если выгружать команды в Telegram уже сейчас, безопасный набор такой:

- `/start` — стартовая настройка
- `/plan` — составить план недели
- `/week` — открыть текущий план
- `/review` — оценить блюда
- `/cancel` — отменить текущий сценарий

## Notes

- В этот документ стоит добавлять только slash-команды, которые пользователь вызывает напрямую.
- Кнопочные действия вроде `Подтвердить неделю`, `Подобрать замену` или `Не предлагать` сюда не входят.
- Перед обновлением списка команд в BotFather нужно сначала обновить этот файл.
